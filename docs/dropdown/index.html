Settings

target
  • Type: string
  • Default: null

Sets an ID of a target dropdown layer.

height
  • Type: int
  • Default: false

Sets dropdown height.

width
  • Type: int
  • Default: false

Sets dropdown width.

animation
  • Type: boolean
  • Default: true

Turns opening and closing animation on and off.
